# How to Install

## Install dependencies
`npm install`

## Setup .env
- Create a .env in the root folder
- Declare vars in .env:
    - PORT
    - HF_TOKEN (got from https://huggingface.co)
    - GOOGLE_MAPS_API_KEY
    - RATELIMIT_WINDOW_MS
    - RATELIMIT_MAX
    - CACHE_TTL_SECONDS

## Run the app
`npm start`
